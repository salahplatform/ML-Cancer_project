# ML-Cancer_project
Built and compared Logistic Regression and K-Nearest Neighbors to classify breast tumors as malignant or benign (Wisconsin Breast Cancer dataset). Used PCA/t-SNE analysis and F-score feature selection, then tuned both models via grid search. Logistic Regression won on all metrics: 95.6% accuracy, 0.993 AUC, 97.2% recall.
